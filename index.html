Data link instructions
  1. Open your Google Sheet → click File → Share → Publish to web
  2. First dropdown: choose your sheet tab. Second dropdown: choose Comma-separated values (.csv)
  3. Click Publish, confirm, and copy the URL it gives you
  4. Open this HTML file in a plain text editor (Notepad on Windows / TextEdit on Mac)
  5. Find the line: const SHEET_CSV_URL = ""; and paste your URL between the quotes
  6. Save the file, then open it in Chrome or Firefox — your dashboard will appear!
